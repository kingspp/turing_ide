# turing_ide
Integrated Development Environment for WPI's turing clusters

## Installation
Takes 10 mins so get a cup of coffee!

```bash
# Clone the repository
git clone https://github.com/kingspp/turing_ide --recursive

# Install IDE
sh install.sh

# Start IDE
sh start.sh 3000

# Stop IDE
sh stop.sh

# Uninstall IDE
sh uninstall.sh
```
